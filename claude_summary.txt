Claude AI 정리 문서
===================

개요
----
Claude는 Anthropic에서 개발한 대화형 AI 어시스턴트입니다.
안전하고 도움이 되는 AI를 목표로 설계되었습니다.

주요 특징
----------
1. 헌법적 AI (Constitutional AI): 안전하고 도덕적인 응답을 위한 훈련 방법
2. 긴 문맥 이해: 대화의 맥락을 잘 파악하고 일관된 응답 제공
3. 다양한 작업 수행: 텍스트 작성, 분석, 코딩, 수학, 창작 등
4. 정직하고 투명한 소통: 모르는 것은 모른다고 명확히 표현

Claude 모델 버전
-----------------
- Claude 1: 초기 버전
- Claude 2: 향상된 성능과 더 긴 입력 처리
- Claude 3 (Haiku, Sonnet, Opus): 각기 다른 성능과 속도의 모델
- Claude 3.5 Sonnet: 최신 고성능 모델

활용 분야
----------
- 텍스트 작성 및 편집
- 프로그래밍 도움
- 연구 및 분석
- 창작 활동 지원
- 교육 및 학습 보조
- 번역 작업

Claude Code
-----------
Claude의 명령줄 인터페이스(CLI) 버전으로, 
개발자들이 터미널에서 직접 Claude와 상호작용할 수 있게 해줍니다.

MCP (Model Context Protocol)
-----------------------------
Claude가 외부 도구 및 데이터 소스와 연결할 수 있게 하는 프로토콜입니다.
이를 통해 파일 시스템, 데이터베이스, API 등과 상호작용이 가능합니다.

장점
----
- 높은 안전성과 윤리적 기준
- 자연스러운 대화 능력
- 다양한 전문 분야 지식
- 창의적 작업 지원
- 정확하고 신뢰할 수 있는 정보 제공

주의사항
--------
- 지식 컷오프 날짜 이후의 정보는 부정확할 수 있음
- 실시간 정보나 최신 뉴스는 제한적
- 의학이나 법률 등 전문 분야는 전문가 상담 권장
- AI 생성 콘텐츠임을 항상 고려해야 함
